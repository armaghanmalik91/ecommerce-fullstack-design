# Ecommerce Full Stack Design

A responsive full-stack ecommerce web application developed as part of the DevelopersHub Corporation Full-Stack Development Internship.

## Project Objective

The objective of this project is to build a fully functional ecommerce web application with a responsive frontend, backend APIs, dynamic product rendering, cart management, authentication, admin product management, and final deployment.

## Week 1 Completed

Week 1 focused on project setup and static frontend development for desktop and mobile views.

### Completed Pages

- Home Page
- Product Listing Page
- Product Details Page
- Cart Page

### Completed Features

- React + Vite frontend setup
- Tailwind CSS styling
- Responsive desktop and mobile layouts
- Navbar with mobile menu
- Product cards
- Product listing grid
- Product search
- Category filter
- Product details route
- Add to cart functionality
- Cart quantity increase/decrease
- Remove product from cart
- Clear cart
- Cart data persistence using localStorage
- Cart count badge
- Basic UI polish and button click feedback
